## 🚀 Getting Started

1. Run the following commands to get started:

   ```bash
   git clone https://github.com/bur4ky/portfolio-website.git
   cd portfolio-website
   bun install
   ```

2. Change the `src/config.ts` file to your information

3. Run `bun dev` to start the server
